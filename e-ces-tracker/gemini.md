# Project Rules

- **Stack:** Laravel 11, Tailwind CSS, Alpine.js, Blade components.
- **Database:** Models already exist. Use Eloquent to bind data to views.
- **Styling:** Use utility-first Tailwind. No custom CSS files unless necessary.
- **Layout:** Extend the `layouts.app` component.
- **Writing Code** Always read the context before modifying.

# UI Rules
- Primary Green: #15803d (bg-green-700)
- Secondary Light Green: #d9f99d
- Layout: Always use Sidebar + Main Content flexbox.
- Sidebar: Width 64, toggleable via Alpine.js.
- Framework: Laravel 11, Tailwind CSS.
